# InsightNotes AI

InsightNotes AI is a full-stack AI-powered note-taking application that helps users organize thoughts, extract insights, and manage tasks automatically.

## 🚀 Features

* Google Authentication using Firebase
* Real-time notes with Firestore
* AI-powered summaries
* Action item extraction
* Automatic tag generation
* AI assistant to query your notes
* Voice-to-text note input
* Dark / Light mode
* Word count and reading time
* Smart search and category filters

## 🛠 Tech Stack

Frontend:

* Next.js
* React
* TailwindCSS

Backend / Infrastructure:

* Firebase Authentication
* Firestore Database
* Vercel Deployment

AI:

* Groq API
* Llama Models
